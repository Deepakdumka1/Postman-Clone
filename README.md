# Postman-Clone

## Setup
```bash
git clone https://github.com/kamalkoranga/Postman-Clone
cd Postman-Clone
cd frontend
npm i
npm run start
cd ..
cd backend
python -m venv env
env\Scripts\activate
pip install -r requirements.txt
flask db upgrade
flask run
```

### Frontend:
- Link: [localhost:3000](http://localhost:3000)
### Backend:
- Link: [localhost:5000](http://127.0.0.1:5000/api)
